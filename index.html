Task 1: Hello World with Variables
Familiarize yourself with declaring variables and strings combined.Create a variable named greeting and set it to "Hello, ".Create another variable named name and set it to your name.Create a third variable that combines or adds a greeting and name.Print the third variable to the console.

Task 2: Basic Math Operations
Practice using numerical operators like addition, subtraction, multiplication, and division.Create two variables num1 and num2 and assign them any number.Create four more variables that store the result of addition, subtraction, multiplication, and modulus of num1 and num2.Print these results to the console.
